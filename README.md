Objective : 
Training a model to best predict loan_status for given data with best performance, highest F1 and ROC-AUC score. Also reducing false positives and over/underfiting to the data.<br>
<br>Procedure : 
1. Clean and prepare data.
2. Feature selection.
3. Model selection.
4. Finetuning.
